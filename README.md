# Energy-Demand-Forecasting
This project analyzes and predicts energy consumption using machine learning techniques. It identifies usage patterns, highlights key factors affecting power demand, and provides insights to improve energy efficiency and optimize resource planning.

# AIM
• Energy Consumption Model using Machine Learning helps predict future energy usage by learning patterns from
  past data like temperature, occupancy, and appliance usage.
• It improves energy efficiency, planning, and resource management by providing accurate consumption forecasts.

# DATASET
* SOURCE=https://www.kaggle.com/datasets/govindaramsriram/energy-consumption-dataset-linear-regression?select=train_energy_data.csv
* FEATURES=Building_Type,          
Square_Footage         ,
Number_of_Occupants    ,
Appliances_Used        ,
Average_Temperature    ,
Day_of_Week            ,

* TARGET=Energy_Consumption

## TECH STACK

* LANGUAGE= python
* LIBRARIES=numpy,pandas,matplotlib,seaborn,scikit-learn
## PROCESS
* Data Cleaning and Preprocessing $ Exploratory Data Analysis (EDA) , Model Training and Testing ,LASSO regression, Model Evaluation and Comparison

## RESULT
* The model successfully predicted energy consumption with good accuracy, showing clear patterns between usage and factors such as time, temperature, and appliance load. The predictions closely matched the actual energy values, proving that the ML model can effectively forecast future consumption and support better energy planning and optimization.

## CONCLUSION
* The Energy Consumption Model effectively identifies usage patterns and accurately forecasts future power demand. The results show that machine learning can be a valuable tool for optimizing energy usage, reducing wastage, and supporting smarter resource management. This model can be further improved with more features, real-time data, and advanced algorithms.
